## Lista algoritmi
| Nome algoritmo                              | Complessità    |
|---------------------------------------------|----------------|
| Capacity Scaling                            | $O(m^2\log C)$ |
| Shortest Augmenting Path                    | $O(n^2m)$      |
| Shortest Augmenting Path + Capacity Scaling | $O(nm\log C)$  |
| Recursive Dinic                             | $O(n^2m)$      |
| Iterative Dinic                             | $O(n^2m)$      |

Il progetto si pone come obiettivo principale l’analisi comparativa approfondita di diverse strategie risolutive per il problema del Massimo Flusso. Lo
studio mira a valutare l’efficienza e la scalabilità di cinque distinti algoritmi, mettendone in luce i punti di forza e
le criticità attraverso un approccio sperimentale.

Dall'analisi è risultato che i migliori algoritmi tra quelli proposti sono Dinic ricorsivo e Shortest Augmenting Path, mentre il peggiore è Capacity Scaling.
